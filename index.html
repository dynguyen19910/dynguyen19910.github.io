Airline Delay Analysis

SQL / Python

Analyzed flight delays by carrier and airport; ranked on-time performance and built a monthly ops summary.

  • Skills: SQL window functions, CTEs, pandas EDA
  • Insight: Weather explains a large share of winter delays in the Northeast
Preview US BTS Data SQL/Python Repo
Sample SQL 
WITH winter AS (
  SELECT carrier, origin, arr_delay
  FROM flights
  WHERE strftime('%m', flight_date) IN ('12','01','02')
),
ranked AS (
  SELECT carrier,
         AVG(arr_delay) AS avg_delay,
         RANK() OVER (ORDER BY AVG(arr_delay)) AS ontime_rank
  FROM winter
  GROUP BY carrier
)
SELECT * FROM ranked LIMIT 10;

import pandas as pd
df = pd.read_csv('flights.csv', parse_dates=['flight_date'])
winter = df[df.flight_date.dt.month.isin([12,1,2])]
summary = (winter.groupby('carrier')['arr_delay']
                .mean().sort_values())
print(summary.head(10))

import pandas as pd
def clean(df):
    for c in df.columns:
        if 'date' in c.lower():
            df[c] = pd.to_datetime(df[c], errors='coerce')
    for c in df.select_dtypes('object'):
        df[c] = (df[c].astype(str).str.strip().str.title())
    return df.drop_duplicates()
