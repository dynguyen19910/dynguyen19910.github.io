Airline Delay Analysis

SQL / Python

Explored delay causes by carrier and airport with SQL and pandas. Produced a monthly performance summary.

  • Skills: SQL (CTEs, window functions), pandas, visualization
  • Insight: Weather drives ~40% of winter delays in the Northeast
Preview Analysis US BTS Data SQL/Python Repo
Sample SQL 
WITH winter AS (
  SELECT carrier, origin, arr_delay
  FROM flights
  WHERE strftime('%m', flight_date) IN ('12','01','02')
)
SELECT carrier, AVG(arr_delay) AS avg_delay
FROM winter
GROUP BY carrier
ORDER BY avg_delay;
Sample pandas 
import pandas as pd
df = pd.read_csv('flights.csv', parse_dates=['flight_date'])
winter = df[df.flight_date.dt.month.isin([12,1,2])]
summary = winter.groupby('carrier')['arr_delay'].mean().sort_values()
print(summary.head(10))

import pandas as pd

def clean_csv(path):
    df = pd.read_csv(path)
    for c in df.columns:
        if 'date' in c.lower():
            df[c] = pd.to_datetime(df[c], errors='coerce')
    for c in df.select_dtypes('object'):
        df[c] = df[c].astype(str).str.strip().str.title()
    return df.drop_duplicates()
